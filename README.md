# ⚽ Dream Football Team 

A dynamic, interactive web application that allows users to assemble their ultimate 11-player football squad from a roster of global superstars. Users must strategically manage their virtual budget to build the best team possible without overspending. 

Built as a milestone project for Day 7 of my #100DaysOfCode challenge.

**🔗 [Live Demo](https://dream-team-musabbir.netlify.app/)**

## 🚀 Features

* **Squad Management:** Users can draft up to 11 players for their dream team.
* **Budget Tracking:** A real-time coin balance system. Users cannot draft a player if they don't have sufficient funds.
* **Interactive Feedback:** Custom alert notifications when a player is successfully added, removed, or if the user hits their budget or roster limit.
* **Responsive Design:** A fully mobile-responsive UI that seamlessly transitions from a desktop grid layout to a stacked mobile view.
* **Claim Free Credit:** A feature to inject more virtual funds into the user's balance to afford premium players.
* **Toggleable Views:** Users can easily switch between the "Available Players" roster and their "Selected Players" dashboard.

## 🛠️ Tech Stack

* **Frontend Framework:** [React.js](https://reactjs.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **UI Components:** [DaisyUI](https://daisyui.com/)
* **Notifications:** [React Toastify](https://fkhadra.github.io/react-toastify/)
* **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
